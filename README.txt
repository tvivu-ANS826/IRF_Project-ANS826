A program lényege, hogy az aktuális koronavírus adatokat nyilvántartja. 
Az úgy fog megvalósulni, hogy létrehoztam egy adatbázist, amiben van 2 tábla található. 
Az egyikben a betegek adatait tároljuk, a másikban a halálozás körülményeit tárolom.
A program képes lesz arra, hogy az adatbázisban található adatokat kiolvassa, és abból egy Excel fájlt generáljon.
Formon lesz 2 gomb, amiből az egyik exportálja az adatbázisban található adatokat egy Excel fájlba, a másik gomb pedig beolvassa egy tetszőleges Excel fájlt.
A beolvasást követően, a program összeveti az adatbázisban illetve az Excelben található adatokat.
Szükség esetén pedig, frissíti az adatbázist.
Ez lehetővé teszi informatikához nem értő személyek számára, hogy az adatbázisba új adatot 
rögzítsenek.
